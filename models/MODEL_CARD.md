# MODEL_CARD — petsense-v0

**Version:** 0.1.0
**Date:** 2026-07-20
**Status:** Synthetic training only — NOT validated on real CSI data

---

## Model Overview

- **Architecture:** 3-head CNN (Conv1D → GlobalAvgPool → 3 classifiers)
- **Input:** `[1, 128, 8]` — batch × window_size × PCA components
- **Outputs:**
  - `presence`: `[1]` — binary (pet present / absent)
  - `species`: `[3]` — human / dog / cat
  - `activity`: `[6]` — walking / running / lying / playing / scratching / jumping
- **Export format:** ONNX opset 17, IR v8
- **Variants:**
  - `petsense-v0.onnx` (185 KB) — float32
  - `petsense-v0-fp16.onnx` (94 KB) — float16 quantized
  - `petsense-v0-int8.onnx` (57 KB) — int8 quantized

## Training

- **Dataset:** Synthetic CSI data generated by `models/generate_synthetic.py`
  - `SyntheticCSIGenerator` produces subcarrier amplitude/phase signals
  - Body mass / speed / distance / gait modulation via `_body_perturbation`
  - NOT real-world WiFi CSI captures
- **Training config:** 46 epochs, `best.pt` checkpoint
- **Optimizer / loss:** Not recorded in checkpoint (`val_metrics=None`, `val_loss=None`)

## Validation

- **Real-world validation: NOT PERFORMED**
- Model runs and produces shape-correct outputs via `onnxruntime-node`
- Inference on zero tensor: `presence≈1.0`, `species=[≈0,≈0,1]` (cat), `activity=[…,0.9999]` (jumping)
- **No per-class precision/recall on real CSI data**

## Limitations

1. **Synthetic-only training** — model has never seen real WiFi CSI signals
2. **No validation metrics** — `val_metrics=None` in checkpoint
3. **No real-world testing** — performance on actual ESP32-S3 CSI captures unknown
4. **Synthetic data bias** — `generate_synthetic.py` uses simplified body perturbation models that may not capture real multipath, fading, or interference
5. **Room classification** — `location` output head is defined but not trained (always returns `null`)

## Intended Use

- Pet tracking through walls using commodity WiFi CSI (2.4 GHz minimum, 5 GHz recommended for cats)
- Edge inference on Raspberry Pi 5 (~3ms per classification)
- MQTT input from 2-3 ESP32-S3 CSI capture nodes
